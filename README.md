# Blendoku

It is a simple implementation of a `colors in a line` game inspired by Blendoku (original game at http://www.blendoku.com/) 

![Screenshot](/screen.png?raw=true )

## How to run

```
$ git clone https://github.com/IvanShamatov/blendoku.git
$ cd blendoku
$ bundle install
$ bin/blendoku
```
